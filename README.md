Together is a web app to organize smaller communities.

## Features
- Task management
- Shared Shopping list
- Debt manager

## Setup

Clone the repository

```
git clone https://github.com/Hiebeler/Community-Frontend.git
```

Install the required packages

```
npm install
```

Run the web app

```
npm run start
```
